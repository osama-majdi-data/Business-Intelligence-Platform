# 📊 Business Intelligence & Customer Analytics Platform

> **An end-to-end SQL & Power BI Business Intelligence solution for executive reporting, product analytics, CRM insights, and data quality auditing.**

---

## 🚀 Project Overview

This project demonstrates how SQL and Power BI can be combined to build a complete Business Intelligence platform.

The project focuses on four business areas:

- 📈 Executive Overview
- 📦 Product Performance
- 👥 Customer Behavior & CRM Analytics
- 🛡️ Data Integrity & Anomalies Audit

---

## 🛠️ Tech Stack

| Tool | Purpose |
|------|----------|
| 🗄️ SQL Server | Data Cleaning & Business Analytics |
| 📊 Power BI | Dashboard Development |
| 🪟 Window Functions | Running Totals, LAG, Ranking |
| 📑 Views | Data Modeling for Power BI |
| 📈 Business KPIs | Executive Reporting |
